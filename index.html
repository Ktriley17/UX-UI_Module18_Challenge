My UX Skills

User Research

User researching is vital to development of a usable website. This consists of gaining quanatative and qualititative through researching trends, surveying, and testing. Through data anaysis you can make cognitive descisions in UX that can be beneficial for the user and brand.

User Interface Design

After user-based, data analysis I'm able to make design decisions that will improve the overall productivity of a website, or improve whatever goal is needing to be met. The design process begins with wireframing which move to web prototypes which both go through user-testing and interation processes.

Web Development

Once a web-redesign is completed in a prototype the development process begins consisting of using HTML, CSS, and JavaScript. This is an important process to keep in mind as a UX/UI designer and laying out a managable design for development.
